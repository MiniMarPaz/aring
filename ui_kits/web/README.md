# Wimbledon.com — Web UI Kit

High-fidelity recreation of Wimbledon's marketing + authenticated site. Components are cosmetic, modular JSX meant to be composed into high-fidelity mocks.

## Open `index.html` to explore:

- **Home** — logged-out hero, latest news rail, scores rail, schedule
- **Scores** — live scores board, Order of Play
- **Player profile** — profile header, stats, match history
- **Ticket** — ballot status

## Components
- `Header.jsx` — sticky top nav with roundel logo, active state, auth CTA
- `HeroChampionships.jsx` — dark-green hero with big Gotham display headline + date strip
- `NewsCard.jsx` — news/feature card with tag + eyebrow
- `LiveScore.jsx` — live scorecard (player / sets / game)
- `OrderOfPlay.jsx` — match schedule list for a given court
- `PlayerHeader.jsx` — player profile top header with seed + country
- `Footer.jsx` — dark footer with partner logos
- `Button.jsx` — primary / secondary / tertiary / danger
- `Tag.jsx` — color-coded content tags
